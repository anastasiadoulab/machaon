Class ExecutionHandler

Description:

Handles executions for sets of tasks that contain extremely long executing processes.

Methods:



__init__()

Description:

Constructor


Arguments:
  • cores: CPU cores that will be allocated for the parallel execution. (int)
  • max_seconds: Maximum waiting time for a process. (int)

Class Attributes:
  • progress_bar [default: None]: X
  • max_wait_time [default: max_seconds]: Maximum waiting time in seconds for a lagging process (int)
  • cores [default: cores]: User-specified maximum processes (int)
  • chunk_size [default: 1000]: The data input to the parallelized processes is divided by this chunk size.
parallelize()

Description:

Executes a function in parallel. The input is divided into chunks, each chunk served in a serial manner.


Arguments:
  • func: The function to be parallelized.
  • entries: The input data for the function that will run in parallel.
Returns:
  • results: The results of all spawned processes.
