Class Evaluator

Description:

This is the evaluation module of Machaon.

Methods:



__init__()

Description:

Constructor


Arguments:
  • viral [default: False]: Whether the data for evaluation are viral or not (This is to prevent the search in the viral data part of local resources. Performance related.) (boolean)

Class Attributes:
  • reference_data [default: defaultdict()]: Evaluation data for the reference protein. (dict) (see load_data())
  • candidate_data [default: defaultdict()]: Evaluation data for the candidate protein. (dict) (see load_data())
  • _aligner [default: StructureAligner()]: StructureAligner object
  • root_disk [default: '']: Current setting of root disk where the third-party data and the target dataset reside. (string)
  • pdb_dataset_path [default: '']: Full PDB directory path (string)
  • viral [default: viral]: Whether the data for evaluation are viral or not (This is to prevent the search in the viral data part of local resources. Performance related.) (boolean)
  • verbose [default: True]: Toggle verbose output (boolean)
  • _max_secondary_structure_length [default: 1500]: Maximum secondary length to be aligned. (int)
  • override_pdb_id [default: '']: Override reference pdb id with another for retrieving Uniprot accession number. Useful for custom PDB files and filenames.

set_pdb_path()

Description:

Sets full PDB directory path


Arguments:
  • pdb_path: Full PDB directory path (string)
load_data()

Description:

Loads relevant data for a protein.


Arguments:
  • pdb_id: PDB ID (string)
  • chain_id: PDB Chain ID (string)
  • refseq_id [default: '']: RefSeq ID (string)
Returns:
  • data: A dictionary with the following contents (at best/if all data are available):
    • '1D': full protein sequence
    • '1DPDB': protein sequence that is present in the PDB file (might contain gap characters due to missing data)
    • '2D': secondary structure sequence of DSSP labels as estimated by PDB file (might contain gap characters due to missing data)
    • 'fingerprint': RDKit object of a molecular fingerprint
    • '5UTR': 5'-UTR sequence
    • 'CDS': CDS sequence
    • '3UTR': 3'-UTR sequence
get_protein_sequence()

Description:

Retrieves full protein sequence from UniProt.


Arguments:
  • uniprot_accession: UniProt Accession identifier (string)
Returns:
  • sequence: Protein sequence (string)
get_viral_information()

Description:

Method used for viral proteins only. Retrieves gene and RefSeq id from UniProt.


Arguments:
  • uniprot_accession: UniProt Accession identifier (string)
Returns:
  • output: A dictionary that has the keys 'geneid' and 'refseqid' and their corresponding values. (empty dict if there is a failure)
load_go_data()

Description:

Reads Gene Ontology information from enrichment data.


Arguments:
  • data_row: Candidate enriched entry (pandas dataframe)
Returns:
  • result: Dictionary with the following keys:
    • 'molecularFunction'
    • 'cellularComponent'
    • 'biologicalProcess'
    Each key has a list of strings with GO term IDs.
align_sequences()

Description:

Performs alignments on currently loaded sequences. (see load_data())


Arguments:
  • sections: The selection of sequence types that are going to be aligned (keys of the loaded data dictionary). (list of strings)
Returns:
  • result: A dictionary with the alignment results that has the sections argument as keys and tupples of (<score>,<identity>,<excluded-gap identity>) as values.
compare_gene_ontology()

Description:

Compares the currently loaded Gene Ontology information. (see load_data())

Returns:
  • result: List that holds a value per GO term type (list of floats) (empty if there are no data available)
calculate_tm_score()

Description:

Performs 3D structural comparison between a reference structure and a candidate one. (Template Modeling Score, TM-Score)

Returns:
  • result: A tupple with (<score>,<candidate structure length>) as values. ((-1,-1) if there is a failure)
