Class PDBHandler

Description:

This is a heavily loaded class that handles low-level functions on PDB files and PDB meta-data requests.

Methods:



__init__()

Description:

Constructor


Arguments:
  • pdb_id [default: '']: PDB ID (string)

Class Attributes:
  • parser: PDB parser (BioPython object)
  • structure_id [default: pdb_id]: Current PDB ID (string)
  • chain_angles [default: defaultdict()]: Dictionary holding information about angles per chain.
    For each chain, the angle data are organized in alternative ways and the following keys exist:
    • 'angles':a tuple of (psi,psi) pairs with None as a possible value in incomplete pairs
    • 'phi'
    • 'psi'
    • 'phiPsiPairs': complete pairs, the ones used by the method.
  • uniprot_accession_number [default: '']: UniProt Accession identifier (string)
  • residue_selection [default: []]: Current selection of residues, one-based indexed numbering (list of integers)
  • domains [default: []]: Current collection of domains (list of strings)
  • points [default: []]: Current collection of atomic coordinates (xyz) (list of tupples)
  • residue_distances [default: []]: Current collection of alpha-carbon inter-residue distances (list of floats)
  • root_disk [default: '']: Current setting of root disk where the third-party data and the target dataset reside
  • verbose [default: True]: Toggle verbose output (boolean)
  • residue_alternative_names [default: ['CYX', 'CYM', 'HID', 'HIE', 'HIP', 'ARN', 'ASH', 'GLH', 'LYN', 'TYM']]: Alternative residue names (list of strings)
  • camera_params_path [default: os.path.join('draw','camera-params')]: Path for the folder containing camera parameters for Open3D visualization
    (see also comments in the implementation of save_visualization() and check example files in the draw/camera-params folder):
    • <PDB ID>_<Chain ID>-alpha-camera-params.json
    • <PDB ID>_<Chain ID>-img-trunc-margins.json (margins in percentages to be truncated from Open3D output)
    • <PDB ID>_<Chain ID>-lineset-camera-params.json
    • <PDB ID>_<Chain ID>-pcd-camera-params.json

get_uniprot_accession_number()

Description:

Retrieves the corresponding UniProt Accession identifier for a PDB chain.
It fetches the mapping with the following ways (in the shown order, lower orders are fallback cases):
  • Loading from the cache
  • Parsing the SEQRES meta-data entry in PDB file
  • Searching in the UniProt local static resources
  • Querying RCSB GraphQL Web Service
Arguments:
  • chain_id: PDB chain ID (string)
  • pdb_id [default: '']: PDB ID (string) [optional, if not provided the current PDB ID (class attribute: self.structure_id) is used]
  • pdb_path [default: '']: Full PDB file path (string) [optional: This is used for obtaining the required information from SEQRES entry.
    If not provided, this way is skipped if it is invoked.]
Returns:
  • self.uniprot_accession_number: Returns the value of the class attribute (string)
