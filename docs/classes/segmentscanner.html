Class SegmentScanner

Description:

This class is dedicated to constrained mode comparisons on segments. Inherits Scanner class.

Methods:

Class Attributes:
  • minimum_residue_range [default: 3]: Minimum length of accepted alignment of segments (int)
  • _column_headers [default: ['structureId', 'b-phipsi', 'w-rdist', 't-alpha', 'alignedRange',]: Column headers of comparison results (list of strings)
  • selected_alignment_level [default: 'mixed']: Options for alignment level: 'primary', 'secondary', 'hydrophobicity', 'mixed'
  • alignment_levels [default: ['primary', 'secondary', 'mixed', 'hydrophobicity']]: The available alignment levels.
  • segments [default: []]: The user-specified segments (list of lists, each list contains residue positions (one-based indexing).

Description:

Extracts features from reference segment.


Arguments:
  • entry: Composite input: list [reference PDB path, residue positions of segment, segment index, reference PDB chain ID, available residues in PDB(one-based indexed)]
Returns:

  • angle_data: psi-psi angle pairs (list of pairs, pair: list of floats)
  • residue_distances: inter-residue alpha-carbon distances (list of floats)
  • triangle_metric: number of triangles (from alpha shape) (int)
