Class ViralRefSeq

Description:

The class that retrieves viral transcript information.

Methods:



__init__()

Description:

Constructor


Arguments:
  • refseq_id: RefSeq ID (string)
  • gene_id: Gene ID (string)

Class Attributes:
  • refseq_id : RefSeq ID (string)
  • gene_id : Gene ID (string)
  • root_disk [default: '']: Current setting of root disk where the third-party data and the target dataset reside. (string)
  • refseq_dataset_path [default: '']: RefSeq data file path(see set_root_disk()) (string)
  • refseq_annotation_path [default: '']: RefSeq annotation file path(see set_root_disk()) (string)
  • sequence [default: '']: Transcript sequence
  • genome_info [default: []]: Current RefSeq annotation data
set_root_disk()

Description:

Sets all the required directories according to the provided root directory path.


Arguments:
  • root_disk: Directory path where the third-party data and the target dataset reside. (string)
fetch_viral_genome()

Description:

Fetches viral genome from RefSeq local resources for the current RefSeq ID (class attribute: self.refseq_id)

fetch_viral_genome_info()

Description:

Fetches genome-level annotations from RefSeq local resources for the current RefSeq ID (class attribute: self.refseq_id)

parse_gene_data()

Description:

Retrieves available coding and non-coding sequences for the current RefSeq ID (class attribute: self.refseq_id)

Returns:
  • data: Dictionary with '5-UTR', 'CDS', '3-UTR' as keys and their sequences as values (depending on the data availability).
retrieve_sequence()

Description:

Retrieves sequences according to provided annotations.


Arguments:
  • gene_info: Annotations (list of pairs, pair: list of int)
Returns:
  • sequence: (string)
fetch_gene_info()

Description:

Retrieves gene-level annotations and sequence (CDS) from RefSeq local resources for the current RefSeq ID (class attribute: self.refseq_id)


Arguments:
  • genome_info: Genome annotations (list of strings)
Returns:
  • parsed: Dictionary with 'CDS' as key and its sequence as the value.
