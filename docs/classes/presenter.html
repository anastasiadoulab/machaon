get_information()

Description:

Retrieves information from UniProt.


Arguments:
  • uniprot_accession: UniProt Accession identifier (string)
Returns:
  • output: Dictionary containing parts of XML formatted UniProt information:
    • 'protein': protein name
    • 'gene': gene name
    • 'organism': origin
    • 'sequence': full protein sequence length
    • 'function': references about functions
    • 'subunit': information about subunits
    (empty dict if there is a failure)

get_lineage_aggregated_information()

Description:

Parses UniProt data for taxonomy information


Arguments:
  • entry: Composite input (list):
    • uniprot_accession: UniProt Accession identifier (string)
    • aggregated_taxonomy [can be None]: Aggregated taxonomy (from recursive calls). (dict)
    • top_roots [can be None]: Top root nodes (from recursive calls). (list of strings)
    • max_depth [can be None]: Current max depth (from recursive calls). (int)
Returns:
  • lineage: Aggregated taxonomy (dict)
  • roots: Top root nodes (list of strings)
  • max_depth: Current max depth (int)
