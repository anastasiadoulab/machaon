Class ConfigurationManager

Description:

This class is responsible of managing the configuration of Machaon's jobs. It also holds the default configurations values which are overriden by user-specified job configurations.

assign_default_root_folder()

Description:

Assigns a default root directory path where source and cached data are stored.


Arguments:
  • path: directory path (string)
parse_console_arguments()

Description:

Parses console arguments and creates a new job configuration.


Arguments:
  • console_arguments: User-provided options.
Returns:
  • configuration: New job configuration (dictionary)
parse_config()

Description:

Parses the master job configuration file (config.yaml)

Returns:
  • collected: A batch of active job configurations (configurations with "ignore : True" directive are ignored)
parse_segment_config()

Description:

Parse segments.yaml file where the user-specified reference segments reside.

Returns:
  • collected: The parsed information of segments. (list of lists, each nested list is a list of residue positions [list of ints]). The segments appear in the order declared in the yaml file. (declarations with "ignore : True" directive are ignored)
  • known: Marked segments by the user as known areas. (list of lists, each nested list is a list of residue positions [list of ints]).
expand_residue_ranges()

Description:

Convert residue ranges declarations to a joint list of residue positions. In case of erroneous presets, the programs exits with code 0.


Arguments:
  • residue_ranges: Ranges of positions, comma-separated. (string)
Returns:
  • residues: List of residue positions, one-based indexing (list of ints).
verify_config()

Description:

Verify that several required aspects of a job configuration have been set. In case of missing presets, the programs exits with code 0.


Arguments:
  • configuration: Configuration dictionary.
  • config_index: Order of job configuration in the yaml file. (int)
