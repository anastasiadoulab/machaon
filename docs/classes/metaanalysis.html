aggregate_go_information()

Description:

Coordinates Gene Ontology-based multiple alignments and stores into a dictionary. This dictionary is also save into a file via object serialization (Pickle package). The structure of the dictionary: alignment_info[<propertyType>][<go_id>][<alignment_level>] and each entry has 'identity', 'mask' and 'alignment' keys. 'mask' entry contains a bitmask of the aligned areas, 'alignment' holds the alignment object (see StructureAligner.get_alignment_positions()) and 'identity' refers to the sequence identity metric.
