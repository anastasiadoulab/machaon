Class StructureAligner

Description:

This class is responsible for the retrieval of protein structures/sequences and RNA sequences, the generation of mixed sequences and performing alignments.

align()

Description:

Performs global sequence alignment: match score=5, mismatch score=-4 similar to EDNAMAT scoring matrix, open gap penalty=-10, gap extension penalty=-0,5,
no end-gap penalty, for protein sequence alignment type, BLOSUM62 matrix is used.


Arguments:
  • reference_sequence: (string)
  • sequence: (string)
  • data_type: 'PROTEIN' (BLOSUM62 scoring is used) or 'OTHER' (string)
  • config [default: {}]: Dictionary for options. Available options:
    • 'removeGaps': gaps are removed from sequences before alignment
    • 'compressGaps': multiple contiguous gaps are squeezed to a single one before alignment
    • 'gapChars': Replace gap char "-" with 2 different characters of choice before alignment (list of string) (recommended since the gap chars inserted by the alignment
      process refer to a misalignment not missing data, it also affects the sequence identity calculation)
Returns:
  • score: Alignment score (float or False if there is a failure)
  • alignment: Aligned sequences (list of string or False if there is a failure)
  • alignment_result: Formatted result of alignment (list of string or empty string if there is a failure)
