Constructor


Class Attributes:
  • debugging [default: False]: True : The method is executed in single core and verbose mode facilitating debugging.
  • max_cores [default: 1]: This value should be set according to the current/expected workload on the side. The maximum value should be the number of the available system's cores minus one, for stability.
  • meta_metrics [default: ['1D-identity', '2D-identity', '5UTR-identity', 'CDS-identity', '3UTR-identity', '3D-score']: The columns of the results that will be used for polar plots.
